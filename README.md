# dev

1. Clonar el archivo .env.template a .env
2. Configurar las variables de entorno
3. Ejecutar el commando ``` npm install ```
4. Levantar las bases de datos con el commando ``` docker compose up -d ```
5. Ejecutar ``` npm run dev ```
